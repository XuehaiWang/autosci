# Neuroscience_003 — 9%

## 任务概述
复现 DELVE（动态特征选择）方法：自底向上的无监督两步框架（局部邻域动态模块识别 → 轨迹图排序），在 11 种方法上做 benchmark（precision@k、分类准确率、伪时间相关性），并用 PHATE 可视化特征选择效果（准确率 0.96，NMI 0.60）。

## 失分根因
1. **方法方向错误**：agent 实现了自顶向下的特征评分（全局参考流形），而 DELVE 是自底向上的（先识别局部动态模块，再做轨迹图排序）。这是方法论上的根本偏差。
2. **基线数量不足**：只对比了 3 种方法（全特征、方差、动态），而 checklist 要求 11 种方法，且使用了不同的评估指标（trustworthiness vs precision@k）。
3. **可视化方法错误**：用 UMAP/PCA 代替 PHATE，无法展示论文要求的相位分离效果。
4. **缺少鲁棒性实验**：噪声注入实验（低 SNR、dropout）完全缺失。

## 数据/方法缺口
- **方法偏差（决定性原因）**：task_plan 没有识别出 DELVE 的"自底向上"特性，实现了相反方向的方法
- **工具缺失**：需要 PHATE 可视化，agent 用 UMAP 代替
